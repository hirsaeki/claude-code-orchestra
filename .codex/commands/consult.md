Claude Code からの相談に回答してください．

## 相談内容

$ARGUMENTS

## 回答方針

### 1. 問題の整理
- 相談の本質を正確に把握
- 不明確な点があれば明確化

### 2. 判断基準の提示
- どのような基準で判断すべきか
- 考慮すべき観点

### 3. 具体的な推奨
- 明確な方向性を示す
- 「場合による」で終わらせない
- 実行可能なアクションを提示

### 4. 注意点
- 見落としやすいリスク
- 事前に確認すべきこと

## 出力形式

簡潔に，かつ実行可能な形で回答してください．

```markdown

{結論を1-2文で}

### 推奨アクション
```
